# 🎓 Student Attendance Tracker – Frontend (React + JavaScript)


## 🚀 Features

- 📋 Mark daily attendance
- 📈 View attendance summary
- 📊 Pie chart visualizations
- 🎯 Filter by date, class, subject
- 🧭 Responsive UI with Bootstrap

## 🛠 Tech Stack

- React.js
- Bootstrap 5
- Axios
- React Router DOM
